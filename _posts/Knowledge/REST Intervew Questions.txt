What are JAXB and JAXRS?

	JAXB (Java Architecture for XML Binding) and JAXRS (Java API for RESTFull Webservices) are Java community process (JCP) standards with multiple implemetations.
		
	JAXB : Defines standards for converting Java objects from/to XML
	JAXRS : Defines standards for runtime API for creation of RESTFull services. JAX-RS, defined in JSR 311).
